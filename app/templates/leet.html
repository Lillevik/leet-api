Leet score api

Score and graph api for the the leet fun api

Here are the different endpoints:
1.

GET

/leet/score/{server_id}
Fetches the score for all users from the given server id. Example: 

{
  "scores": [
    {
      "month_score": 9,
      "nick": "nick1",
      "score": 4,
      "streak": 0
    },
    {
      "month_score": 7,
      "nick": "nick2",
      "score": 27,
      "streak": 0
    },
    {
      "month_score": 6,
      "nick": "nick3",
      "score": 177,
      "streak": 1
    },
     ...
}
2.

GET

/leet/graph/{server_id}
Fetches the graph data for all nicks belonging to a server. Example: 

{
  "Nick1": {
    "graph": [
      {
        "date": "2017-06-16",
        "streak": 0
      },
      {
        "date": "2017-06-17",
        "streak": 1
      },
        ...
    {
  "Nick2": {
    "graph": [
      {
        "date": "2017-06-16",
        "streak": 0
      },
      {
        "date": "2017-06-17",
        "streak": 1
      },
        ...
}
3.

GET

/leet/graph/{server_id}/{nick}
Fetches the graph data for given nick belonging to a server. Example: 

{
  "Nick": {
    "graph": [
      {
        "date": "2017-06-16",
        "streak": 0
      },
      {
        "date": "2017-06-17",
        "streak": 1
      },
        ...
}
4.

GET

/leet/graph/average/{server_id}
Fetches the average streak for each day of the given server. Example: 

{
    'graph': [
    {
      "avg_streak": 0.46153846153846156,
      "date": "2017-12-23"
    },
    {
      "avg_streak": 0.3076923076923077,
      "date": "2017-12-24"
    },
    {
      "avg_streak": 0.23076923076923078,
      "date": "2017-12-25"
    },
    ...
  ]
}
5.

GET

/leet/graph/average/{server_id}/{nick}
Fetches the average streak, highest streak and number of days its based on for the given user from the given server. Example: 

{
  "graph": [
    {
      "avg_streak": 0.7473684210526316,
      "count": 190,
      "max_streak": 5
    }
  ]
}
More endpoints might come in the future when leet cash is implemented.
